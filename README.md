# SukokuSolverAndroid
An Implementation of Sudoku game for android 13 that can solve the game itself.

## Available Options
- Importing game boards from .sdk files
- start/stop solver
- reset game board

## SDK file format
```
2..1.5..3
.54...71.
.1.2.3.8.
6.28.73.4
```
